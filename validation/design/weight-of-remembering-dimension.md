# Dimension 01 — The Weight of Remembering

## Construction thesis

This is not a finite room and not a dense 3D reconstruction. It is a seeded dimension manufactured from one artwork, a semantic world manifest, and many cheap independently addressable effects.

The existing room model initializes the dimension so the world remains compatible with Confluence camera, category, color, and runtime concepts. `Dimension.fromRoom(...)` then extends that seed into:

- six depth layers;
- eight named world regions;
- four traversable currents;
- sixty independently animated or interactive nodes;
- a persistent dimensional law;
- a machine-readable browser validation bridge.

## Manufacturing strategy

The source artwork provides visual substance. The runtime adds life through inexpensive Canvas 2D operations rather than thousands of meshes:

- parallax image offset;
- stars, city lights, lanterns, glows, and pulses;
- drifting framed memories;
- swaying chain links;
- petals and moths;
- mist and depth fog;
- portals and anchors;
- animated filament paths;
- pointer-responsive depth;
- pause/resume and region inspection;
- a world-construction map.

## Boundary

This PR establishes the reusable `Dimension` class and the first image-seeded runtime. It does not yet include audio, touch-to-collect interactions, portal traversal, authored camera journeys, image segmentation into true cutout layers, or publishing to the production room sequence.
